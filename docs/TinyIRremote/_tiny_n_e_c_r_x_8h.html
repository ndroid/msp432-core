Detailed Description

Minimal IR receiver for NEC protocol.

Receives IR protocol data of NEC protocol using pin change interrupts. NEC is the protocol of most cheap remote controls for Arduino.

No parity check is done! If callback function is enabled, on a completely received IR command the user handler function is called in Interrupt context but with interrupts being enabled to enable use of delay() etc. !!!!!!!!!!!!!!!!!!!!!! Functions called in interrupt context should be running as short as possible, so if you require longer action, save the data (address + command) and handle them in the main loop.
